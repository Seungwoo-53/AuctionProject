
<h1>이벤트 경매</h2>
<a href="https://www.event-auction.kro.kr/" target="_blank">사이트 바로가기</a>

<h2>프로젝트 기획 동기</h3>
<div>
  <ul>
    <li>중고나라, 당근마켓 같은 중고거래 플랫폼에 경매시스템을 적용하면 어떨까?</li>
  </ul>
</div>

<h2>프로젝트 핵심기능</h2>
<div>
  <ul>
    <li>웹소켓 기반 실시간 경매 입찰</li>
    <li>경매 아이템을 인기순, 가격순, 시간순으로 정렬기능</li>
    <li>입찰 랭킹 히스토리, 조회수 [세션으로 중복체크]</li>
    <li>마이페이지 [입찰 참여 했던 경매아이템 목록]</li>
    <li>관리자계정 [경매아이템 관리, 회원 관리]</li>
  </ul>
</div>
